## 附录：技术选型理由

| 技术 | 选型理由 |
|-----|---------|
| uni-app | 一套代码支持多端小程序，开发效率高 |
| FastAPI | Python生态，异步支持好，AI集成方便 |
| MySQL | 成熟稳定，小程序后端标配 |
| Redis | 高性能缓存，支持会话管理和热点数据 |
